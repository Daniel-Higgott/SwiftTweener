Supported Types

  • addType(toType:toArray:isNumeric:)

    Method to add support to custom Types.

    Declaration

    Swift

    public func addType<T>(toType: @escaping ToTypeBlock<T>, toArray: @escaping ToArrayBlock<T>, isNumeric: Bool = false)

    Parameters

    toType

    An instance of ToTypeBlock that converts values from Double’s Array to declared Type.
    toArray

    An instance of ToArrayBlock that converts values from declared Type to Double’s Array.
    isNumeric

    A Bool that declares if Type is a Numeric Type, Swift by default casts 1.0 as Double Type and 1 as Int Type, Tweener casts those Doubles and Ints to Type automatically.
