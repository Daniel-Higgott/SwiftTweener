  • init(target:duration:ease:delay:from:to:completion:)

    Creates a descartable Tween.

    Declaration

    Swift

    public convenience init(target: T,
                        duration: Double = 1.0,
                        ease:@escaping Equation = Ease.none,
                        delay:Double = 0.0,
                        from: [PartialKeyPath<T> : Any]? = nil,//Optional
                        to: [PartialKeyPath<T> : Any]? = nil,
                        completion: TweenHandler? = nil)

    Parameters

    target

    Required, target instance to perform animations.
    duration

    Optional, Time duration of tween in seconds.
    ease

    Optional, Easing equation for animation curve, the default value is linear.
    delay

    Optional, Time to delay after start this animation.
    from

    Optional, collection of pair type KeyPath and Any, this sets the intial animation values, if isn’t defined the engine takes the current values from target as start.
    keys

    Optional, collection of pair type KeyPath and Any to animate a property with the desired end values, Tweens never start if not defined.
    replace

    Optional, Boolean value that indicates if the engine removes existing tweens for same property.
    completion

    Optional, A code block to execute when animation has finished.

    Return Value

    A new Tween instance .
